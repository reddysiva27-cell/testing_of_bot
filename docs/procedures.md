# Procedures

## Employee Onboarding
1. Create employee record in the **HRMS system**.
2. Assign reporting manager and department.
3. Provide system access (email, VPN, HR portal).
4. Schedule induction training within the first week.
5. Ensure ID card and workstation are ready before Day 1.

---

## Expense Reimbursement
1. Collect receipts for all expenses.
2. Submit receipts via the **Finance Portal** within 30 days of expenditure.
3. Manager reviews and approves the claim.
4. Finance team processes reimbursement within **7 business days**.
5. Payment is credited to the employee’s salary account.

---

## Leave Application
1. Log in to the **HR Portal**.
2. Navigate to **Leave Management → Apply Leave**.
3. Select leave type (Casual, Sick, Earned).
4. Enter start and end dates.
5. Submit for manager approval.
6. Track status in the portal dashboard.

---

## Password Reset
1. Navigate to **Settings → Security → Reset Password**.
2. Enter your current password and new password.
3. Confirm via email verification link.
4. Password reset is complete once the link is validated.

---

## Escalation Process
1. Raise issue with your **manager**.
2. If unresolved, escalate to **HR**.
3. If still unresolved, escalate to **Head of Department**.
4. Document escalation steps in the **Issue Tracker**.
